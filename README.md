# Posts-Viewer
📰 Premium Newspaper App

A modern responsive newspaper-style web app built using HTML, CSS, and JavaScript.

## 🚀 Features
- Fetch posts from API
- Click post to view comments
- Search functionality
- Error handling + Retry button
- Smooth animations
- Responsive design
- Premium UI with hover effects

## 🛠️ Technologies
- HTML5
- CSS3 (Flexbox, Animations)
- JavaScript (Fetch API)

## 📂 Project Structure
- index.html
- style.css
- script.js

## ⚡ How to Run
1. Download or clone repo
2. Open index.html in browser

## 🌐 API Used
https://jsonplaceholder.typicode.com/
